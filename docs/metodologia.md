# Metodologia

O desenvolvimento do projeto seguirá as seguintes etapas:

## 1. Revisão bibliográfica

Análise de estudos sobre manguezais e variáveis ambientais críticas.

## 2. Documentação (Durante um período de Março a Outubro)

Realizar a documentação referentes aos nossos estudos e as nossas adições dentro do projeto durante o ano.

## 3. Definição de requisitos

- Funcionais: coleta, transmissão, visualização
- Não funcionais: desempenho, confiabilidade

## 4. Desenvolvimento do hardware

- Integração de sensores
- Testes de coleta de dados

## 5. Desenvolvimento do software

- Backend para recebimento e armazenamento
- Frontend para visualização

## 6. Integração

- Comunicação entre hardware e software

## 7. Testes

- Validação dos dados coletados
- Simulação de cenários críticos

## 8. Documentação

- Registro de todas as etapas no repositório
