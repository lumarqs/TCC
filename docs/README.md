# Sistema de Monitoramento de Manguezais

## Descrição

Sistema de monitoramento ambiental em tempo real voltado para manguezais, com foco na medição de oxigênio dissolvido como indicador de risco ecológico.

## Objetivo

Desenvolver uma solução tecnológica capaz de auxiliar na preservação da biodiversidade por meio da coleta e análise contínua de dados ambientais.

## Tecnologias

- ESP32 / Arduino
- Node.js ou Python
- Banco de dados (Firebase ou PostgreSQL)
- Dashboard web

## Estrutura

Ver pasta /docs para documentação completa do projeto.

## Impacto

O sistema contribui para a detecção precoce de condições ambientais críticas, auxiliando na prevenção da degradação e possível extinção de espécies em manguezais.
